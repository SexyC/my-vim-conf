# My Vim configuration file and plugins

### Inlcuded files:

1. vimrc
2. plugins

### Install Method:

**via curl**

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/SexyC/my-vim-conf/master/install)"
```

**via wget**

```bash
sh -c "$(wget https://raw.githubusercontent.com/SexyC/my-vim-conf/master/install -O -)"
```
